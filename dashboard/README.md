# Dashboard del Operador — Fase post-acuerdo

Este directorio es el placeholder para el dashboard web de Podium.

## Cuándo se construye
Después de cerrar el acuerdo con el inversor (Wick, Jay, o independiente).

## Qué incluirá
- Lista de partidos analizados, predicciones y resultados
- Tasa de acierto por liga, mercado y período
- Vista limitada para el inversor (solo métricas, no tecnología)
- Vista de canales activos (suscriptores, última publicación)

## Stack planificado
- React o HTML+JS puro (según complejidad necesaria)
- Datos desde Supabase (API REST o supabase-js)
- Hosting: Vercel (gratuito en tier básico)
- Dominio: opcional (~$12/año)
